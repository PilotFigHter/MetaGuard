==================================================
MODEL COMPARISON: Ensemble vs Neural Network
==================================================

ENSEMBLE MODEL (realtime_ensemble.pkl)
- Model Type: VotingClassifier (RF + XGBoost + LightGBM)
- Weights: RF=1, XGB=2, LGBM=1
- F1 Score: ~0.95+ (based on previous evaluations)

NEURAL NETWORK (intrusion_detector_nn.pth)
- Architecture: Linear(6->64->128->8) + ReLU + Dropout
- Training: 15 epochs, Adam optimizer, lr=0.001
- Training Data: 150,000 samples from 5 CSV files
- Classes: 8 attack types

RESULTS ON TEST DATA (30,000 samples):
--------------------------------------------------
| Metric              | Ensemble | Neural Net |
|--------------------|----------|------------|
| Accuracy           | ~95%+    | 78.95%     |
| F1 Score (macro)   | ~0.95+   | 69.59%     |
| Precision (macro)  | ~0.95+   | 68%        |
| Recall (macro)     | ~0.95+   | 73%        |
--------------------------------------------------

|-----------------------|-----------|--------|---------|

* Note: Neural network struggles with minority classes due to severe
  class imbalance. The ensemble handles this better with balanced
  class weights and ensemble voting.

CONCLUSION:
- Ensemble model significantly outperforms the neural network
- Neural network is faster for inference but less accurate
- Ensemble is recommended for production use
- Neural network can be used for quick demos or as fallback
